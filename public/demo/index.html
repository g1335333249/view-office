View Office Preview

Office 文件预览服务

通过文件 URL 或文件流创建预览会话，前端引用 SDK 后即可把 doc、docx、xls、xlsx、ppt、pptx、pdf、txt 等文件嵌入到业务系统中预览。

页面引用

把 SDK 引入业务页面，然后指定容器和文件来源。

URL 预览

服务端会下载 URL 文件并转交给 Collabora 预览。

ViewOffice.preview({
  container: "#viewer",
  source: {
    type: "url",
    url: "https://example.com/demo.docx",
    fileName: "demo.docx",
    fileType: "docx"
  }
});

文件流预览

适合业务系统中用户本地上传后直接预览。

ViewOffice.preview({
  container: "#viewer",
  source: {
    type: "file",
    file: file,
    fileName: file.name,
    fileType: file.name.split(".").pop()
  }
});

文件清理

关闭预览或创建新预览时会主动删除服务端临时文件。浏览器异常关闭时，服务端每日清理超过保留时间的文件。

ViewOffice.cleanup(session);
输入文件地址或上传文件后开始预览
